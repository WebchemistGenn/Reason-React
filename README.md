# Reason React

> Reason React를 이용하여 TODO 프로젝트를 만들어 봅니다.

## Install Project

```bash
$ yarn create react-app [project-name] --scripts-version reason-scripts
```

## Setting

```json
...
// bsconfig.json
"reason": {
  "react-jsx": 3
}, // jsx ver 3 이용하기
"sources": ["src", "src/components"], // js 컴파일 폴더 추가
...
```
